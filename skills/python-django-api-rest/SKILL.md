# Python + Django + API REST

## Quando usar
Para implementar ou evoluir módulos backend com Django, APIs REST e organização por domínio.

## Checklist
1. Definir contrato REST (request/response/status code).
2. Isolar regras de domínio em serviços/use-cases.
3. Manter serializers/validações explícitas.
4. Adicionar testes de unidade e integração para endpoints críticos.
5. Garantir observabilidade mínima (logs e erros claros).
