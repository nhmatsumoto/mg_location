# Docker + Redes + Runtime

## Quando usar
Para padronizar ambientes com Docker Compose, rede entre serviços e confiabilidade operacional.

## Checklist
1. Definir Dockerfile enxuto e reproduzível.
2. Configurar compose com healthchecks e dependências.
3. Revisar rede/ports/DNS entre containers.
4. Padronizar variáveis de ambiente e segredos.
5. Validar startup, logs e readiness.
