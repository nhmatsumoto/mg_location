# Execução de Engenharia de Software

## Quando usar
Para coordenar implementação técnica com previsibilidade e qualidade.

## Checklist
1. Quebrar demanda em tarefas pequenas e rastreáveis.
2. Definir contratos entre módulos e integrações.
3. Implementar incrementalmente com validação contínua.
4. Cobrir fluxo crítico com testes.
5. Entregar com documentação de operação e rollback.
