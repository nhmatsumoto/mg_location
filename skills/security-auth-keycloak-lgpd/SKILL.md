# Segurança + Autenticação + Keycloak + LGPD

## Quando usar
Para implementar login seguro, controle de acesso e conformidade de privacidade.

## Checklist
1. Mapear fluxos de autenticação (OAuth2/OIDC).
2. Proteger endpoints por roles/scopes.
3. Validar armazenamento/rotação de segredos e tokens.
4. Classificar dados pessoais e aplicar minimização (LGPD).
5. Registrar auditoria e incidentes de segurança/privacidade.
