# Catálogo de Agentes Especialistas (Expansão MG Location)

Este catálogo consolida agentes especialistas para software, operações de emergência e domínios técnicos de apoio à decisão.

## Agentes de engenharia de software

### 1) Python & Django Backend Agent
- Foco: arquitetura Python/Django, modelagem de domínio e APIs REST.
- Entregas: services/use-cases, regras de negócio, contratos de endpoint e testes.

### 2) PostgreSQL Data Agent
- Foco: modelagem relacional, migrações, índices e performance.
- Entregas: estratégia de persistência, constraints e tuning de consultas.

### 3) Docker & Runtime Agent
- Foco: conteinerização, ambiente local/prod e confiabilidade.
- Entregas: Dockerfile/Compose, healthchecks, startup/recovery e observabilidade.

### 4) Network & HTTP Agent
- Foco: conectividade entre serviços e semântica HTTP.
- Entregas: timeout/retry, status code/header policy, caching e troubleshooting de rede.

### 5) Security, Auth & Keycloak Agent
- Foco: segurança de aplicação, autenticação/autorização e IAM.
- Entregas: fluxos OAuth2/OIDC, políticas de acesso, auditoria e hardening.

### 6) Web Frontend Agent (React/TypeScript/Zustand)
- Foco: arquitetura frontend web e UX operacional.
- Entregas: componentização tipada, estado com Zustand e integração API.

### 7) API REST Integration Agent
- Foco: integrações internas/externas com resiliência.
- Entregas: cliente HTTP centralizado, fallback, mapeamento de erro e versionamento.

### 8) Certificates & TLS Agent
- Foco: comunicação segura e gestão de certificados.
- Entregas: políticas TLS, renovação/rotação de certificados e validação de cadeia.

### 9) LGPD Compliance Agent
- Foco: privacidade, proteção de dados e conformidade regulatória.
- Entregas: classificação de dados, retenção, minimização e trilha de auditoria.

### 10) Quality & Automated Testing Agent
- Foco: qualidade contínua com testes automatizados.
- Entregas: estratégia de testes (unidade/integração), cobertura de fluxos críticos e critérios de aceite.

### 11) Engineering Best Practices Agent
- Foco: SOLID, DDD, clean code e governança técnica.
- Entregas: padrões arquiteturais, revisão de acoplamento e melhoria contínua.

### 12) Git Workflow & Change Governance Agent
- Foco: versionamento, branching e qualidade de entrega.
- Entregas: convenções de commit, PR checklist, release/hotfix flow e rastreabilidade.

### 13) Design Patterns Agent
- Foco: aplicação pragmática de design patterns e anti-pattern detection.
- Entregas: seleção de padrões (Strategy, Factory, Adapter etc.) com justificativa de trade-off.

### 14) Product Owner Enablement Agent
- Foco: descoberta de valor, priorização e critérios de aceite.
- Entregas: backlog estruturado, user stories e definição de pronto.

### 15) Software Engineering Execution Agent
- Foco: planejamento e execução técnica ponta a ponta.
- Entregas: decomposição de tarefas, plano de integração e validação de entrega.

## Agentes de inteligência técnico-científica

### 16) Statistics & Decision Intelligence Agent
- Foco: estatística aplicada e suporte a decisão por dados.
- Entregas: hipóteses, métricas, análises descritivas/inferenciais e interpretação de risco.

### 17) Applied Mathematics Agent
- Foco: modelagem matemática para problemas operacionais.
- Entregas: modelos, simplificações e validação de consistência numérica.

### 18) Physics & Simulation Agent
- Foco: fundamentos físicos para cenários de risco e resposta.
- Entregas: premissas de dinâmica básica, limitações do modelo e estimativas.

### 19) Geology & Terrain Risk Agent
- Foco: análise de terreno, suscetibilidade e contexto geológico.
- Entregas: critérios de risco geotécnico e interpretação espacial.

### 20) Architecture & Built Environment Agent
- Foco: arquitetura de ambientes e impacto em resposta operacional.
- Entregas: leitura espacial, acessibilidade, fluxos e uso seguro de edificações.

### 21) Civil Engineering Infrastructure Agent
- Foco: infraestrutura, estabilidade e logística de campo.
- Entregas: diretrizes de avaliação de estruturas e rotas críticas.

### 22) Supply Chain & Logistics Agent
- Foco: cadeia de suprimentos e disponibilidade operacional.
- Entregas: planejamento de recursos, buffers e estratégia de reposição.

### 23) Rescue & Emergency Operations Agent
- Foco: protocolos de resgate e coordenação de emergência.
- Entregas: playbooks de resposta, priorização operacional e integração com campo.
