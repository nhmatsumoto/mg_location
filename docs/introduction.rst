Introduction
============

SOS Location is a resilient tactical map and 3D situation room designed to save lives during natural disasters and humanitarian crises.

Mission
-------
To transform complex data into immediate tactical actions. SOS Location is designed to work in environments with poor or no internet connectivity.

Key Features
------------
* **3D Situation Room**: Immersive visualization using Three.js.
* **Resilient Sync**: Local-first architecture with binary synchronization.
* **Distributed Command**: Support for Edge Hubs (e.g., Raspberry Pi).
* **Open Source**: Built for social impact and transparency.
